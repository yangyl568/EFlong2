## 💡 深度思考

> **"不要在忙碌中迷失，保持独立思考的能力。"**

### 💼 关于工作

1. 总是说公司很忙，业务很忙。那自己怎么办？
2. 项目中有遇到哪些难点？
3. 一年又过去了你给公司留下了什么？
4. 请清晰明了的讲下你负责的业务。

### 🌟 关于自我

1. 有哪些亮点、成就？
2. 你在通过这份工作获得了什么？
3. 你有哪些不足之处，接下来应该怎么解决？

## 🚀 如何提升效率

每做一件事情的时候，都要花一分钟思考下，该任务可以分成多少个模块，然后模块间如何被统筹。
如何做好规划，整理思路，避免把整块时间变得支离破碎。

---

## 📈 如何升值加薪

1. **明确评判标准**: 根据职级划分任务难度和分量。超出预期则 `+` 分，符合预期则 `平` 分，不符合预期则 `-` 分。
2. **超越预期**: 在完成本职工作的基础上，主动发现并解决团队痛点。
