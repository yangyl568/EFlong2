# 资深 Web 前端求职方案（基于当前背景）

## 一、目标与定位

- 目标岗位：资深 Web 前端开发
- 技术栈：JavaScript、Vue 全家桶（熟练）、React（会用）、Node.js
- 目标：快速进入面试流程并拿到 offer

## 二、项目经历梳理（3–5 个重点项目）

**目标**：形成简历里的“可量化贡献 + 技术深度 + 业务影响”

**建议模板（每个项目 5 行内）**：

- 项目名称 / 时间 / 角色
- 背景：业务目标与痛点
- 职责：核心模块/架构/关键技术
- 结果：性能、转化、成本、稳定性等数据指标
- 技术点：复杂表单、权限体系、性能优化、工程化、可视化等

**行动建议**：

1. 先列全量项目清单（10–15 个），再筛 3–5 个
2. 每个项目强调你做了什么、带来什么价值

## 三、简历优化（结构清晰 + 指标导向）

**建议结构**：

1. 个人信息（精简）
2. 技能栈（框架/工程化/性能/协作分层）
3. 项目经历（按影响排序，不按时间）
4. 工作经历（项目集合 + 负责模块）
5. 其他（技术文章 / 开源 / 证书）

**推荐句式**：

- 将 X 模块性能从 3.2s 优化到 1.4s，首屏耗时下降 56%
- 重构权限体系，覆盖 8 个业务线，配置错误率下降 70%
- 推动 CI/CD + 自动化测试，发布效率提升 3 倍

## 四、两周面试准备计划（每天 2–3 小时）

### 第 1 周：夯实基础 + 强化项目

- D1：JS 基础（作用域、闭包、原型链）
- D2：异步（Promise、事件循环、async/await）
- D3：浏览器（渲染流程、缓存、性能指标）
- D4：工程化（Webpack/Vite、构建优化）
- D5：框架原理（Vue 重点，React 了解）
- D6：项目复盘 + 面试话术
- D7：算法基础（数组/字符串/双指针）

### 第 2 周：专题突破 + 模拟面试（细化）

- D8：性能优化 & 监控
- D9：组件设计/状态管理
- D10：CSS/布局/兼容性
- D11：Node/后端协作（接口、鉴权）
- D12：系统设计（复杂表单、权限、低码场景）
- D13：模拟面试（项目深挖）
- D14：补弱项 + 总复盘

## 五、下一步可选项

1. 你提供 3–5 个项目关键词，我帮你整理成简历项目描述
2. 我直接产出一份完整简历文本（含自我介绍）
3. 根据目标城市/薪资/行业，细化两周计划和投递节奏

## 六、两周计划细化（面向资深 Web 前端）

### 第 1 周：核心基础 + 项目话术

- D1：JS 基础复盘（原型链、闭包、this、作用域）+ 2 道手写题
- D2：异步与事件循环（Promise/async/await）+ 1 题手写 Promise
- D3：浏览器与性能（渲染、缓存、首屏、性能指标）+ 总结 5 个优化点
- D4：工程化（Webpack/Vite、构建优化、CI/CD）+ 项目中一处优化案例
- D5：Vue 原理（响应式、diff、组件通信）+ 项目中的架构设计复盘
- D6：React 要点（Hooks、状态管理、性能）+ 对比 Vue 的优劣
- D7：项目复盘日（准备 2 个完整项目“STAR 话术”）

### 第 2 周：专题突破 + 模拟面试

- D8：组件设计与状态管理（Vuex/Pinia、可复用性）+ 组件拆分原则
- D9：复杂场景（权限、表单、低码/可配置）+ 设计方案输出
- D10：性能优化体系（监控、埋点、长列表、资源加载）+ 指标量化
- D11：Node.js 协作（接口规范、鉴权、中间层）+ 中台协作经验
- D12：系统设计题（中后台框架、权限体系、流程编排）+ 画架构图
- D13：模拟面试（项目深挖 + 追问演练）+ 自我复盘
- D14：补弱项 + 总结（形成个人能力亮点清单）

## 七、投递节奏（两周冲刺版）

### 第 1–3 天：简历与渠道准备

- 产出 1 版主简历 + 1 版岗位定制简历
- 准备 20–30 家目标公司清单（按优先级分 A/B/C）
- 完成 1 版自我介绍（1 分钟 + 3 分钟）

### 第 4–10 天：集中投递 + 高频面试

- 每天投递 6–10 家（优先 A，再 B）
- 每天安排 1–2 场模拟面试/自测
- 每天输出 1 份面试复盘（问题清单 + 修正点）

### 第 11–14 天：收敛与补强

- 聚焦面试反馈中的弱项（集中突破 2–3 个核心点）
- 复投重点公司 + 内推优先
- 面试话术与项目成果再包装
